🍜 今天吃什么留言板

Vue 3 + Spring Boot + MySQL 全栈项目

📦 技术栈

Vue 3 Vite Axios Spring Boot 3 Spring Data JPA MySQL 8

🚀 启动步骤

  • 1. 数据库 — 执行 database/init.sql 创建数据库
  • 2. 后端 — 修改 backend/src/main/resources/application.yml 中的数据库密码，然后运行：
cd backend
mvn spring-boot:run
  • 3. 前端 — 安装依赖并启动：
cd frontend
npm install
npm run dev
  • 4. 访问 — 打开浏览器访问 http://localhost:5173

📁 项目结构

Saying Board/
├── database/init.sql      ← 数据库脚本
├── backend/               ← Spring Boot 后端
│   ├── pom.xml
│   └── src/main/java/com/foodboard/
├── frontend/              ← Vue 3 前端
│   ├── package.json
│   └── src/
└── main.html              ← 本文件（项目说明）
